# Student Management System (Java)

## 🚀 Features
- Add student
- View students
- Update student details
- Delete student records
- File handling (data persistence)
- GUI using Java Swing

## 🛠️ Technologies Used
- Java
- Swing (GUI)
- File Handling
- OOP Concepts

## ▶️ How to Run
1. Compile: javac *.java
2. Run: java StudentGUI
